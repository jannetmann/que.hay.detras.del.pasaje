# ¿Qué hay detrás del pasaje?
Entender las condiciones laborales de los conductores de microbús en México es pieza clave para la movilidad de millones de personas.

El objetivo de este recurso es **humanizar la experiencia**, mostrando de manera inmersiva cómo son sus jornadas laborales, qué tensiones atraviesan, qué tensiones generan y cómo se viven las relaciones conflictuales derivadas de sus condiciones de trabajo.

Al explorar e investigar qué hay detrás de un conductor de microbuses, nos enfrentamos a tristes situaciones actuales. En esta experiencia buscamos transmitir un poco de lo que entendimos.
